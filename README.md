### Spotly — Travel & Location Discovery App
`Python` `FastAPI` `PostgreSQL` `PostGIS` `SQLAlchemy` `AWS S3` `Firebase`

Backend for a mobile location discovery and AR souvenir collection app,
built against a database of ~2.7 million OpenStreetMap places.

Highlights:
- Zoom-adaptive map clustering with a three-tier fallback architecture
  (SQL aggregation → pre-computed DB views → Python bucketing)
- Dual-database setup: transactional main DB + read-optimized fast-places DB
- JWT auth with token versioning, blacklisting, and Firebase OAuth
- AR souvenir collection system with bonus mechanics
- Scheduled background jobs for map archive generation and token cleanup
